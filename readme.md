# Genshin novel reader go server

This is server for Genshin novel reader using Gin (golang), support latest frontend version.

## code structure 

```
├── config
│   ├── config.go
│   └── config.json
├── db
│   └── db.go
├── docker-compose.yml
├── go.mod
├── go.sum
├── main.go
├── middleware
├── readme.md
├── routes
│   ├── index
│   ├── ping
│   ├── routes.go
│   └── user
└── templates
    └── welcome.html
```
